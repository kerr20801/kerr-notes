# Machine Learning Cheat Sheet

## 1. 迴歸 (Regression) - 預測連續數值

* **線性迴歸 (Linear Regression)**
* **適合情境**：預測房價、銷售額預估、評估廣告投入與營收的關係。


* **多項式迴歸 (Polynomial Regression)**
* **適合情境**：預測非線性成長的數據，例如傳染病擴散速度、特定物理資源的消耗曲線。


* **正規化迴歸 (Ridge / Lasso Regression)**
* **適合情境**：欄位（特徵）極多且彼此高度相關時，用來防止模型過度擬合（Overfitting）。Lasso 還能順便幫你篩選重要特徵。



## 2. 分類 (Classification) - 預測離散類別

* **邏輯迴歸 (Logistic Regression)**
* **適合情境**：最基礎的二分法。例如：垃圾郵件分類（是/否）、信用卡交易是否欺詐。


* **隨機森林 (Random Forest)**
* **適合情境**：顧客流失預測、銀行貸款審核。適合處理包含類別與數值的混合資料，且不容易看錯特徵。


* **XGBoost / LightGBM**
* **適合情境**：各大資料科學競賽常客。適合處理結構化的表格數據，例如電商點擊率（CTR）預測、使用者行為精準分類。


* **支持向量機 (SVM)**
* **適合情境**：樣本數不多但特徵很複雜的情況。例如：文字情感分析、生物基因分類。


* **單純貝氏 (Naive Bayes)**
* **適合情境**：資料量小且多為文字時。例如：新聞文章的自動標籤分類。



## 3. 非監督式學習 (Unsupervised Learning) - 無標籤找規律

* **K-Means 分群**
* **適合情境**：客戶特徵細分（客群分析）、 exploratory data profiling，把行為相似的人群揪出來。


* **DBSCAN 分群**
* **適合情境**：地理座標資料分析（如外送熱區、犯罪熱點預測）。能自動識別不規則形狀的群集，並排除雜訊。



## 4. 降維 (Dimensionality Reduction) - 精簡特徵、視覺化

* **主成分分析 (PCA)**
* **適合情境**：資料欄位太多導致計算太慢時，在預處理階段進行資料壓縮；或是消除欄位之間的重疊影響。


* **t-SNE / UMAP**
* **適合情境**：把幾百個特徵的高維度數據，壓縮到 2D 或 3D 空間，畫成圖表來直觀觀察資料的聚集趨勢。



## 5. 深度學習與序列模型 (Deep Learning & Sequence Models) - 處理非結構化與時間資料

* **卷積神經網路 (CNN)**
* **適合情境**：所有與影像相關的任務。例如：瑕疵檢測、物件識別、醫學影像診斷。


* **長短期記憶網路 (LSTM / GRU)**
* **適合情境**：與時間順序、前後文相關的資料。例如：伺服器異常流量的時間序列預測、氣象預報。


* **Transformer**
* **適合情境**：現代 AI 的核心。例如：大型語言模型（LLM）、機器翻譯、文本摘要、程式碼自動補全。
