# AI 讓你的行政助理：30 分鐘速成工作坊

> 目標對象：完全不懂程式的行政人員
> 時長：30 分鐘
> 核心訊息：你不需要懂技術，只需要學會怎麼說話

---

## AI 是什麼？一句話說清楚

AI 不是搜尋引擎，是你的「全能實習生」。

| | 搜尋引擎 | AI 助理 |
|--|---------|---------|
| 給你什麼 | 連結，你自己去找答案 | 直接幫你寫、幫你整理、幫你想 |
| 你的角色 | 自己讀、自己整理 | 給方向、下指令、最後確認品質 |

**重點：你不需要懂技術，只需要學會「怎麼說話」**

---

## 萬用 Prompt 公式

記住這四步，什麼都能問：

```
1. 你是誰    → 你是一位專業的行政助理
2. 做什麼    → 幫我寫一封請假申請信
3. 給條件    → 語氣正式、不超過 100 字
4. 給素材    → 我明天要請病假
```

**範例完整句子：**
> 「你是行政助理，幫我寫一封請假信，語氣正式不超過100字，我明天要請病假。」

---

## 實戰一：Email / 公文

直接複製這個 Prompt，把【】換成你的情況：

```
你是一位專業的行政人員。
幫我寫一封【信件類型】的 Email。

收件人：【對象】
主旨：【主旨】
重點：【你要說的事】

要求：
・語氣【正式 / 親切】
・長度不超過 150 字
・結尾加上署名
```

適用情境：催款信、系統維護通知、婉拒廠商報價、會議邀請

---

## 實戰二：會議記錄

```
STEP 1  開會時用手機錄音，或隨手記關鍵字
STEP 2  把逐字稿或關鍵字貼進 ChatGPT
STEP 3  下 Prompt（見下方）
STEP 4  確認輸出，修改後直接發出
```

**Prompt：**
```
以下是會議的逐字稿/記錄，請幫我整理成：
1) 會議重點（3-5點）
2) 決議事項
3) 待辦任務（含負責人）

格式要清楚，可以直接對外發送。

【在這裡貼上你的會議記錄或逐字稿】
```

---

## 實戰三：資料整理 / 摘要

| 需求 | Prompt |
|------|--------|
| 長文件摘要 | 「幫我把以下文件整理成重點摘要，不超過 5 點，每點一行」 |
| 資料分類 | 「以下是一份清單，幫我依照【類別】分類，並用表格呈現」 |
| 比較表格 | 「幫我比較以下幾個選項的優缺點，用表格呈現，欄位包含：價格、功能、適合對象」 |
| 改寫潤稿 | 「以下是我寫的草稿，幫我改得更流暢、專業，保留原意，語氣【正式/親切】」 |

> 以上所有 Prompt 最後都加上：「內容如下：」再貼你的資料

---

## 常見地雷

| 地雷 | 解法 |
|------|------|
| AI 會亂講，數字/日期/人名可能是假的 | 永遠自己確認，它亂講時態度很自信所以更危險 |
| 答案太長沒重點 | 加一句「請用條列式，不超過5點」 |
| 機密資料別貼上去 | 公司內部敏感資訊先刪掉再問 |
