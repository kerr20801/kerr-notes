# 選股系統裡，ML 佔幾%？比你想的少很多

> 寫於 2026-05-05。這是我在建了一個跑了一年的台股自動選股系統後，對「AI 選股」這件事的誠實回顧。

---

## 大家想像的 AI 選股

```
歷史資料 → 深度學習模型 → 預測明天漲的股票 → 買進
```

這個畫面很吸引人。輸入資料，模型輸出答案，系統自動執行。

現實是什麼樣子？

---

## 實際系統的結構

我的系統選股流程分三階段：

**Stage 1：籌碼篩選（規則制）**
- 外資買賣強度 >= 門檻
- 外資連買天數 >= 門檻
- 股價 > 10 元
- 成交量 > 門檻

**Stage 2：防呆排雷（規則制）**
- 排除過熱股（5日漲幅過大）
- 排除高波動股
- 排除黑名單

**Stage 3：供應鏈評分（規則 + 主題標籤）**
- 在熱門供應鏈主題內 → +2 分
- 外資連買強度高 → +1 分
- 外資預測模型信心高 → +1 分
- 分數 >= 4 → BUY

ML 在這個流程裡的位置：Stage 3 裡的外資預測模型，貢獻一個加分項。

---

## 為什麼是規則制，不是純 ML

**原因一：可解釋性**

規則出了 BUY，我知道為什麼。
模型出了 BUY，我只知道它覺得這支漲的機率比較高。

當一支股票表現很差，我需要知道是哪個條件判斷錯，才能修系統。純 ML 沒辦法給我這個。

**原因二：ML 的誠實數字**

我做了幾個實驗：

| 模型 | AUC | 說明 |
|------|-----|------|
| 外資翻轉預測 v1 | 0.896 | 假的。今天收盤資料預測今天的外資，資料洩漏 |
| 外資翻轉預測 v2 | 0.859 | 用昨天資料預測今天。但主要是昨天外資方向主導，量價貢獻很小 |
| 純量價模型 | 0.606 | 誠實數字。用昨天量價預測明天漲跌，接近隨機 |

AUC 0.606 代表：用昨天的量價預測明天漲跌，比亂猜好一點點。

這不是我的模型爛，這是量價特徵本身的資訊量就這麼多。

**原因三：外資籌碼是落後指標**

外資連買第 3 天，你才知道外資連買了 3 天。進場時的實際狀況是「外資已經買了 3 天」，不是「外資接下來會繼續買」。

外資籌碼是確認工具，不是預測工具。用它來選股，等於確認外資正在進場，而不是預測外資要進場。

---

## ML 真正有用的地方

不是「預測哪支股票明天漲」，而是：

**1. 盤中即時訊號**

相對大盤強度（今天漲而大盤跌）+ 量比（今天量是均量的幾倍）——這兩個特徵在盤中就能即時計算，比等晚上外資資料更快。

盤中訊號的時效性遠大於準確率的邊際提升。

**2. 異常偵測**

某支股票的成交量突然是平均的 5 倍，且外資大買，但股價幾乎沒動——這個組合很罕見，值得特別關注。

這類異常偵測用規則很難寫，ML（Isolation Forest）反而適合。

**3. 風險模型**

市場整體波動率高的時候，選股門檻應該提高。這個「環境判斷」可以用 ML 量化，比手動設閾值更動態。

---

## 一個結論

選股系統裡，ML 的角色是「輔助確認」，不是「主要決策」。

主要決策靠規則：哪些股票符合進場條件。
ML 輔助：在符合條件的股票裡，哪些額外加分，以及整體市場環境如何。

這不是因為 ML 不夠好，而是：

> **規則 + 人的判斷 + ML 輔助，比純 ML 更可靠。**

因為純 ML 系統出錯的時候，你不知道從哪裡修起。

## 附：外資連買分層勝率（真實資料，2026 年 4-5 月）

| 外資連買天數 | 勝率 | 平均報酬 |
|------------|------|--------|
| 3-4 天 | 52.4% | +1.75% |
| 5-7 天 | 48.1% | +0.92% |
| 8-9 天 | 41.2% | -0.15% |
| 10 天以上 | 31.3% | -0.39% |

外資連買越久，不代表報酬越好。高 fc 的股票往往是「外資已經買很多了」，上漲空間反而有限。

這個反直覺的結果，讓我修改了評分系統。
