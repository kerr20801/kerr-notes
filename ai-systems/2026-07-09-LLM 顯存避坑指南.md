💥 避坑指南：為什麼 12GB 顯存跑不動 14GB 的大模型？

在規劃本地端 AI 推理伺服器時，最常出現的一個迷思是：「我買了 RTX 4070 12GB 或 RTX 3060 12GB，是不是就能跑滿 12GB 甚至 14GB 的模型？」

答案是：不行，而且速度會慢到讓你懷疑人生。

本文將以目前最多人使用的 Ollama 為例，並針對三張主流顯卡（RTX 3060 12G、RTX 3060 Ti 8G、RTX 4070 12G），解析本地部署 LLM 時的「顯存陷阱」與效能真相。

📊 顯存的三分天下鐵則

很多人算 VRAM 只看 .gguf 檔案有多大，但實際上 Ollama 運行時的顯存消耗公式如下：

Total VRAM = 📦 模型權重 (Weights) + 🧠 KV Cache (上下文) + ⚙️ 框架開銷 (Overhead)

📦 模型權重 (Weights)： 這是靜態的。例如一個 14B 的 Q4 模型，光是載入就會吃掉約 8.5GB。

🧠 KV Cache： 這是動態的「吃 RAM 怪獸」。為了不重複計算過去的 token，模型必須把 Context（對話紀錄/長文章）暫存起來。上下文越長，吃掉的 VRAM 就越多（通常需要 1GB ~ 2GB 以上）。

⚙️ 框架開銷 (Overhead)： Ollama (基於 llama.cpp) 啟動 CUDA 與保留運算緩衝區，大約需要 1GB 左右的保底空間。

結論：你的 VRAM 必須大於「模型大小 + 至少 2GB ~ 3GB 的餘裕」，否則 Ollama 就會採取「降速求生」策略。

💀 Ollama 顯存不足的「降速求生」模式

不同於其他框架會直接崩潰（OOM），Ollama 非常聰明，當它發現 GPU 顯存不夠時，會啟用 CPU Offloading（卸載）。

機制： 假設模型有 40 層，Ollama 發現 VRAM 只能塞下 25 層，它就會把剩下的 15 層塞到你電腦的系統記憶體 (CPU RAM) 裡。

代價（卡頓真相）： 系統記憶體 (DDR4/DDR5) 的頻寬遠低於 GPU VRAM，且資料必須頻繁跨越 PCIe 匯流排。這會導致模型生成速度從原本順暢的 30~40 tok/s，瞬間暴跌成 3~5 tok/s 的「幻燈片模式」。

特徵： 模型能跑，不會報錯，但每生一個字都要等半天，失去互動價值。

🎯 實戰：三張主流顯卡的「真正甜蜜點」

根據上述鐵則，我們來盤點這三張卡的真實戰力：

1. RTX 3060 Ti (8GB VRAM) —— 被 8GB 封印的算力

硬體特性： 核心算力不錯，頻寬也有 448 GB/s，但 8GB VRAM 是致命傷。扣除系統佔用，實際可用僅剩 ~7GB。

甜蜜點： Llama-3.1-8B (Q4_K_M) 或 Gemma-2-9B (Q4_K_M)

避坑警告： 絕對不要妄想跑 14B 模型（如 Qwen2.5-14B）。因為 14B 模型需要 8.5GB，Ollama 會被迫把大量運算丟給 CPU，導致空有 3060 Ti 的算力卻只能跑出 3 tok/s 的龜速。

2. RTX 3060 (12GB VRAM) —— 窮人 LLM 神卡

硬體特性： 雖然核心與頻寬 (360 GB/s) 不如 3060 Ti，但 多出來的 4GB VRAM 讓它成為平民神器。

甜蜜點： Qwen-2.5-14B (Q4_K_M) 或 DeepSeek-R1-Distill-14B

實戰表現： 14B 模型 (8.5GB) + KV Cache + Overhead 剛好能 100% 塞進 12GB VRAM 裡。因為不需要 CPU Offloading，它可以穩穩跑出約 15~20 tok/s 的實用速度，體驗完爆 3060 Ti。

3. RTX 4070 (12GB VRAM) —— 速度王者，但容量受限

硬體特性： 算力極強，GDDR6X 頻寬達 504 GB/s。但 VRAM 依舊只有 12GB。

甜蜜點： 與 3060 12G 相同，最適合跑 14B 級別模型，但速度更快（可達 30~40+ tok/s）。

避坑警告： 有些人以為 4070 很強，就去跑 Gemma-4-27B 或 Phi-3.5-MoE 42B。由於權重高達 16GB~24GB，Ollama 一樣必須把大量層數卸載到 CPU。結果就是：4070 跑 27B 模型的速度，甚至會降到跟 CPU 算力差不多的 8~11 tok/s，完全浪費了這張卡的價值。

💡 總結

在 Ollama 的世界裡，「容量 (VRAM) 決定你能跑多聰明的模型，算力與頻寬決定你跑得多快」。

選模型： 模型權重檔案大小，最好不要超過你 GPU 總顯存的 70% ~ 75%。

選硬體： 如果你的目標是 14B 等級的中大型模型，買二手 RTX 3060 12G 的體驗，絕對比買效能更強但只有 8G 的 RTX 3060 Ti 還要好得多！
